## 1. openCV 라이브러리 설치

1. **가상환경 활성화 후**
   ```bash
   pip list
   ```
   - 설치된 라이브러리 목록 확인
   - 필요시 ```python -m pip install --upgrade pip``` 입력 후 pip 버전 업그레이드

2. **openCV, Numpy, Matplotlib 라이브러리 설치**
   ```bash
   pip install opencv-python
   pip install numpy
   pip install matplotlib
   ```

3. **라이브러리 설치되었는지 확인**
